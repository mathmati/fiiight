# Try Dropping These — Curated MUGEN/Ikemen Content

Personal test list for the drag-and-drop loader. All links checked 2026-07-12.

## How to zip it

- **Characters:** drop a zip whose contents merge as `chars/<name>/...`, or a zip with the plain character folder (containing the `.def`/`.sff`) at its root. Wrapper folders are auto-stripped by the loader, so a zip *containing* the char folder also works.
- **Stages:** same idea — `stages/<name>.def` + `.sff` at the root, or a folder holding the `.def`/`.sff`.
- **Heads up:** the loader takes **zips only**. Several downloads below are `.7z`/`.rar` — extract, then re-zip the character/stage folder.
- Most downloads already contain exactly one char/stage folder, so "unzip → rezip folder" (or drop as-is if already .zip) is all you need.
